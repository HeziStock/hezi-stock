# Stock Tracker — שיפורים ומוצר למכירה

רעיונות לשיפור ולמוצר שאנשים ירצו לקנות.

---

## כבר בוצע ב-GUI

- **Header** עם שם המוצר וסלוגן
- **כפתור ראשי** "Update now" בולט
- **סטטוס ברור** ("Updated at HH:MM" / "Updating… Please wait.")
- **כרטיסים** (Latest insight, Consider entering, Consider exiting)
- **הגדרות** — שמירה גם כש"market movers" פעיל (בלי סימבולים)
- **ריווח וצבעים** — רקע, כותרות, מראה מקצועי

---

## שיפורים מומלצים למוצר

### חוויית משתמש (UX)

| שיפור | תיאור |
|--------|--------|
| **און־בורדינג** | מסך ראשון עם "התחל" + הסבר קצר (מה המוצר עושה, איך מתעדכנים הנתונים). |
| **טיפים מובנים** | Tooltip או טקסט עזר ליד "Update now" / "Daily schedule" (למשל: "מעדכן מובילי שוק פעמיים ביום"). |
| **הודעת הצלחה** | אחרי "Update now" — הודעה קצרה בדשבורד ("הדוח נשמר בתיקיית Reports") במקום רק בסטטוס. |
| **רענון אוטומטי** | אחרי עדכון מוצלח — אפשרות לרענן את הטבלאות אוטומטית כל X דקות (אופציונלי). |
| **עברית / RTL** | תמיכה בממשק בעברית (תרגום, RTL) לקהל ישראלי. |

### עיצוב ומיתוג

| שיפור | תיאור |
|--------|--------|
| **לוגו** | אייקון ליד "Stock Tracker" ב־header (קובץ .ico ל־Windows). |
| **תמונת נושא** | בחירת ערכת צבעים (כהה/בהיר) או תבנית צבעים קבועה. |
| **גרסה** | הצגת גרסה בחלון "About" או בתחתית ההגדרות (למשל v1.0). |

### פיצ'רים למוצר בתשלום

| שיפור | תיאור |
|--------|--------|
| **רישיון** | מפתח רישיון / מנוי (חודשי/שנתי), בדיקה מקומית או מול שרת. |
| **הגבלת משתמשים** | גרסה חינמית: X עדכונים ביום; גרסה בתשלום: ללא הגבלה + דוחות מתקדמים. |
| **דוחות PDF** | ייצוא דוח יומי/שבועי ל־PDF עם לוגו ומיתוג. |
| **התראות מותאמות** | התראה כשמניה מסוימת עוברת X% (רק בגרסה פרימיום). |
| **היסטוריה גרפית** | גרף מגמה לפי ימים (למשל: מובילי עלייה/ירידה לאורך זמן). |

### טכני ואמינות

| שיפור | תיאור |
|--------|--------|
| **טיפול בשגיאות** | הודעות ברורות כשהרשת נכשלת או ש־Yahoo מחזיר ריק (למשל "נסה שוב בעוד דקה"). |
| **לוג** | קובץ לוג (למשל `logs/stock_tracker.log`) לדיבוג ולתמיכה. |
| **עדכון אוטומטי** | בדיקה אם יש גרסה חדשה (למשל קובץ version.json באתר) והצגת "גרסה חדשה זמינה". |
| **גיבוי הגדרות** | ייצוא/ייבוא של config.json מגרסה לגרסה או בין מחשבים. |

### שיווק ומכירה

| שיפור | תיאור |
|--------|--------|
| **דף נחיתה** | אתר קצר שמסביר מה המוצר, צילומי מסך, מחיר, כפתור "קנה עכשיו". |
| **תמחור** | גרסה חינמית מוגבלת + גרסה בתשלום חד־פעמי או מנוי. |
| **תשלום** | חיבור ל־Stripe / PayPal / ביט (לפי קהל היעד). |
| **התקנה שקטה** | מתקין (installer) ל־Windows עם קיצור דרך ושילוב ב־Task Scheduler. |

---

## סיכום עדיפויות

1. **קצר טווח:** און־בורדינג, לוגו/גרסה, טיפול בשגיאות ו־tooltips.
2. **בינוני:** רישיון בסיסי, דוח PDF, היסטוריה גרפית.
3. **ארוך טווח:** דף נחיתה, תשלום, מתקין, עדכונים אוטומטיים.

אם תרצה, אפשר ליישם צעד־אחר־צעד את הנקודות מהטבלאות (למשל: קודם און־בורדינג ו־About עם גרסה).
