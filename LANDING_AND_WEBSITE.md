# איך להריץ את האתר עם דף נחיתה

## מה יש עכשיו

- **`/` (ראשי)** — דף נחיתה (מכירה): כותרת, יתרונות, למי זה, איך זה עובד, כפתורי "כניסה לדשבורד" ו"ליצור קשר".
- **`/app`** — הדשבורד של HEZI STOCK (המלצות, Run now, הגדרות, דוחות).

כשאתה מריץ **Start HEZI STOCK.bat** ופותח **http://127.0.0.1:5000** — תראה קודם את דף הנחיתה. לחיצה על "כניסה לדשבורד" או "כניסה לאפליקציה" תעביר ל־**http://127.0.0.1:5000/app**.

---

## שינוי כתובת "ליצור קשר"

בקובץ **`templates/landing.html`** יש כפתור "ליצור קשר" שמקשר ל־`mailto:your@email.com`. החלף את `your@email.com` בכתובת המייל שלך.

---

## להעלות לאינטרנט (אפשרויות)

### 1. להריץ על שרת משלך (VPS / בית)

- מתקין Python על השרת, מעלה את כל תיקיית הפרויקט.
- מריץ: `python app.py` (או עם gunicorn: `gunicorn -w 4 -b 0.0.0.0:5000 app:app`).
- מגדיר דומיין (למשל `hezi-stock.com`) שמצביע לכתובת השרת.
- מומלץ: להעמיד מאחורי **nginx** עם **HTTPS** (Let's Encrypt) ו־reverse proxy ל־פורט 5000.
- בהגדרות האפליקציה: להגדיר **סיסמה לפורטל** (Settings) כדי שאף אחד לא ייכנס בלי הרשאה.

### 2. דף נחיתה נפרד (רק שיווק)

- אם הדשבורד רץ רק אצלך (לא באינטרנט), אפשר להעלות **רק** את דף הנחיתה לאתר:
  - ב־**Netlify** / **Vercel** / **GitHub Pages**: יוצרים פרויקט חדש, מעלים קובץ HTML סטטי (מעתיקים את התוכן מ־`landing.html` ומחליפים את הכפתורים בקישור לאתר הדשבורד שלך או ל־"ליצור קשר").
  - ב־**Wix** / **WordPress**: בונה דף עם אותו תוכן ומקשר כפתור "כניסה לדשבורד" לכתובת שבה הדשבורד רץ (אם העלית אותו) או ל־טופס יצירת קשר.

### 3. הכל יחד (נחיתה + אפליקציה)

- מעלה את כל הפרויקט לשרת (כמו בסעיף 1).
- מגדיר דומיין + HTTPS.
- אז:
  - **example.com/** = דף נחיתה
  - **example.com/app** = דשבורד
- משאיר סיסמה ב־Settings לכל מי שנכנס מ־/app.

## עיצוב דף הנחיתה

העיצוב נמצא ב־**`templates/landing.html`** (CSS בתוך הקובץ). אפשר לשנות:

- צבעים — במשתני ה־CSS בראש (`:root`).
- טקסטים — ישירות ב־HTML (כותרות, פסקאות, כפתורים).
- גופן — כרגע Heebo (עברית); הקישור ל־Google Fonts בראש הקובץ.

אם תרצה דף נחיתה נפרד (קובץ HTML סטטי) להעלאה ל־Netlify/Wix בלי Flask — אפשר ליצור קובץ **`landing_standalone.html`** שמכיל את כל התוכן והעיצוב, והכפתור "כניסה לדשבורד" יקשר לכתובת שתגדיר (למשל `https://your-app.com/app`).
